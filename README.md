# Sai Agro Foods Ecommerce Website

Premium React/Vite storefront for Sai Agro Foods, an edible oil and agricultural products brand from Pasupathipalayam, Karur, Tamil Nadu.

## Features

- Responsive ecommerce storefront
- Product catalog, filters, quick view and product detail pages
- Cart with localStorage persistence
- Checkout and thank-you flow
- About, process, recipes, FAQ and contact pages
- WhatsApp ordering links
- Brand-specific animations and visual signatures

## Setup

```bash
npm install
npm run dev
```

Open `http://localhost:5173`.

## Build

```bash
npm run build
```
